生成二维码的组件。

## API

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| text | 按钮文案 | String | '二维码生成' |
| size | 二维码大小 | Number | 150 |
| colsize | input框，按钮的布局  | Number | 6 |
