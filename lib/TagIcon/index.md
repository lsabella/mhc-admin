倒计时组件。

## API

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| spin | 是否有旋转动画 | boolean | false |
| type | 图标类型 | String | - |
| text |  tag中显示的文本内容 | String | -|
| color |  标签颜色 | String | -|
